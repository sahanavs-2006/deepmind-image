# BrandForge AI 🚀

BrandForge AI is a next-generation design platform that generates an entire, cohesive brand identity in seconds. Instead of spending weeks manually drafting strategy and designing assets, BrandForge utilizes the cutting-edge capabilities of the **Gemini 3.1 API** to forge a unified brand—from the core strategy down to the visual marketing assets.

## ✨ Features & Gemini API Integration

This project was built to push the limits of the Gemini 3.1 model suite. Here is how we leverage the different models:

1. **Structured Brand Strategy (`gemini-3.1-pro-preview`)**
   - **How it works:** We use Gemini 3.1 Pro with strict `responseSchema` (JSON Structured Outputs) to deterministically generate a cohesive brand name, tagline, color palette, typography recommendations, and detailed image generation prompts based on user input.

2. **Multimodal "Thinking" Analysis (`gemini-3.1-pro-preview`)**
   - **How it works:** We utilize the model's high thinking mode (`thinkingLevel: ThinkingLevel.HIGH`) to analyze reference images provided by the user. The model breaks down the visual style, colors, and key elements to inform the rest of the brand generation process.

3. **Asset Generation & Iteration (`gemini-3.1-flash-image`)**
   - **How it works:** Using the exact prompts generated by the strategy step, we rapidly generate a suite of assets (Logo, Website Hero, Packaging, Ad Banners, Social Posts) using Gemini's flash image models. We also use reference-based image editing to refine these assets.

## 💻 Tech Stack

- **Frontend:** React 19, React Router v8, Tailwind CSS v4, Framer Motion
- **Backend:** Express.js (Node.js) with Vite Middleware
- **AI/ML:** `@google/genai` SDK

## 🛠️ Run Locally

**Prerequisites:** Node.js (v18+)

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file in the root directory and add your Gemini API Key:
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```

3. Start the development server (starts both the backend and Vite frontend):
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000`.
